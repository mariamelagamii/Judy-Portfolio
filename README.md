# Judy Marketing Portfolio

## التشغيل
استخدمي Live Server داخل VS Code وافتحي `index.html`.

## تعديل المشاريع
عدّلي الملف:
`assets/data/projects.json`

كل مشروع يحتوي على:
- `id`
- `name`
- `category`
- `icon`
- `description`
- `longDescription`
- `image`
- `website`

مهم: تشغيل الصفحة مباشرة عبر `file://` قد يمنع تحميل JSON، لذلك استخدمي Live Server.
